Lesson 2 Guide

Vocabulary:

  • Coordinate Plane: The numbering system that gives each position in a plane an assigned coordinate .
  • Coordinate: Numbers that indicate the position of a point on the coordinate plane.
  • X Coordinate: The “X” value, or measure of the horizontal, left right position. When looking at the coordinate plane, the horizontal line.
  • Y Coordinate: The “Y” value, or measure of the vertical, up down position. When looking at the coordinate plane, the vertical line.

Blocks Used/Introduced Today:

  • This code block moves your sprite to the specified X and Y coordinate. You cannot see the movement visually.

  • This code block moves a sprite left or right by changing the X coordinate value.

  • This code block moves a sprite up or down by changing the Y coordinate value.

  • This code block moves sprite to the specified X and Y coordinate. You can see the movement visually.

  • This code block instructs the sprite to glide to the X, Y location of your Mouse Pointer.

Exercise 1: Mastering the Coordinate Plane

In this exercise, you will understand and master the coordinate plane.


Below is an image of the coordinate plane.

Observe how there are many intersection of lines. In the coordinate plane, we can identify specific locations> thanks to these intersections of lines.


To label our coordinate points, we look at the numerical value the lines correspond to.


First, we look at the left-right, horizontal position. This is known as the X coordinate.


Then, we look at the up-down, vertical position. This is known as the Y coordinate.

Now you know how to label points! Let's learn about trends and patterns of the coordinate plane.


Before we start, remember that the X coordinate is left-right, while the Y coordinate is up-down.

Exercise 2: Using the Coordinate Plane in Scratch

In this exercise, you will apply your knowledge of the coordinate plane to create movements in Scratch.

Use the "Change X by __" block to move the sprite horizontally by changing the X coordinate of the sprite.


Play around with the numbers to figure out the rules to make the sprite move left, and the rules to make the sprite move right.


Use the "Change Y by __" block to move the sprite vertically by changing the Y coordinate of the sprite.


Play around with the values in these blocks to see how they affect the sprite's movement on the coordinate plane. Figure out the rules to moving the sprite up, or down!

Exercise 3: Usages of "move" and "glide"

In this exercise, you will explore the differences between "move" and "glide" blocks in Scratch.


Drag the "Go to X: __ Y: __" block to the workspace, under the when green flag clicked block.


Set the X and Y values to move the sprite to different locations on the stage. Observe how the sprite jumps directly to the specified coordinates.

Keep experimenting to figure out the general trends for specific positions. For example, for anything on the right side of the backdrop, the X coordinate is a big positive value.


Use the "Glide __ seconds to X: __ Y: __" block to make the sprite smoothly move to a specific location on the stage over a specified period of time.

Try different X, Y coordinates and timing values to see how the sprite's movement changes.


Use the "Glide to mouse pointer" block to make the sprite glide to wherever the mouse pointer is located. This is particularly useful for interactive projects where the sprite follows the user's mouse movements.
