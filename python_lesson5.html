Exercise 1: Introduction to Lists

Lists are similar to variables in that they store values, but unlike variables, lists can store many variables of different types. Lists are collections with an unlimited amount of space.

Example of a List:

my_list = [1, "apple", 3.14, True]

Lists are used to store things in an organized manner, often storing similar items or items used for similar purposes.

Exercise 2: Declaring Lists

                
Lists consist of elements, which are the individual items inside a list.

                
To declare a list, use hard brackets [ ] and separate elements with commas.

                
Example of Declaring a List:

                
groceries = ["Milk", "Donuts", "Bread", "More donuts"]

                
As you can see, each element is separated by a comma, and lists can hold all types of elements, including strings, integers, and floats.

Exercise 4: List Functions

                
You can add, remove, replace, and call (get its value) elements in a list using various functions.

                
To call an element:

                
LISTNAME[INDEX_LOCATION]

                
To get the index of an element:

                
LISTNAME.index(ELEMENT)

                
To add an element:

                
LISTNAME.append(ELEMENT)

                
LISTNAME.insert(INDEX_LOCATION, ELEMENT)

                
To remove an element:

                
LISTNAME.pop(INDEX_LOCATION)

                
LISTNAME.remove(ELEMENT)

                
To replace an element:

                
LISTNAME[INDEX_LOCATION] = ELEMENT

Exercise 5: The "in" Operator

                
The in operator is used in if statements to check if a value is inside a string or a list.

                
Example:

                
groceries = ["milk", "donuts", "cookies"]

                
To check if "donuts" is in the list:

                
if "donuts" in groceries:
        print("It is in groceries")
else:
        print("it's not there")
