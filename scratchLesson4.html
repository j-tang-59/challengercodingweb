Lesson 4 Guide

Vocabulary:

  • Loop: A loop is a way to repeat a set of instructions multiple times until a certain condition is met or forever.
  • Conditional: A conditional is a statement that performs different actions based on whether a certain condition is true or false.
  • Forever Loop: A forever loop is a type of loop that repeats the same set of instructions endlessly, without stopping.
  • While Loop: A while loop is a type of loop that keeps repeating a set of instructions as long as a certain condition is true.
  • If Statement: An if statement is a type of conditional that executes a set of instructions only if a certain condition is true.

Blocks Used/Introduced Today:

  • The While Loop. Use this block to create the conditional to where the while loop will execute.

  • The Repeat Loop/For Loop. Use this block to create a loop that repeats the code inside the number of times listed.

  • If Conditional. If the conditional is met, the code inside will execute.

  • If Else Conditional. Allows you to program the conditional if the conditional is not met.

  • This block lets your conditional be checked. The computer will check if the key you selected has been pressed or not.

Exercise 1: Using Loops

A loop is used when you need to repeat an action multiple times. There are different types of loops for different scenarios.


The best time to use a loop is when you know an action needs to be repeated, but you don't want to write the same code over and over again.


Repeat Loop: Use the "Repeat __ times" block when you know exactly how many times you want to repeat an action.


Forever Loop: Use the "Forever" block when you want an action to continue indefinitely until you manually stop it.

Tutorial: Creating a Simple Maze Game

Follow these steps to create a simple automated maze game using Scratch:


Step 1: Draw the Maze Backdrop

Start by creating your own maze backdrop. You can draw the maze directly on the backdrop using the Scratch editor. Make sure the maze has a clear path from start to finish.


Step 2: Add the Sprite

Select a sprite that will navigate through the maze. Position the sprite at the starting point of your maze.


Step 3: Move the Sprite with "Move __ Steps"

Drag a "when green flag clicked" block to the workspace. Add a "move __ steps" block below it. Set the number of steps to move your sprite forward. This will move the sprite in the direction it is facing.


Step 4: Rotate the Sprite with "Turn __ Degrees"

After moving forward, add a "turn __ degrees" block to rotate the sprite. This will allow it to navigate corners and turns in your maze.


Step 5: Use "Wait __ Seconds" to Control the Speed

To slow down the movement and make the animation smoother, add a "wait __ seconds" block between movements. Adjust the timing as needed to make the sprite navigate at the desired speed.


Step 6: Add a "Forever" Loop to Keep Moving

To keep the sprite moving continuously through the maze, wrap the movement and rotation blocks in a "forever" loop. This will ensure the sprite keeps navigating the maze until it reaches the end.


Step 7: Test and Adjust

Click the green flag to test your maze game. Watch how the sprite moves through the maze. Adjust the movement, rotation, and timing as needed to make sure the sprite follows the correct path through the maze.


Step 8: Add Finishing Touches

Add any finishing touches, such as sound effects or a message when the sprite reaches the end of the maze. You can also customize the sprite's appearance or add decorations to the maze.
