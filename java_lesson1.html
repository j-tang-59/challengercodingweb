Common Java Errors
  • Error: "; expected"
    Cause: Missing semicolon at the end of a statement
    Fix: Add a semicolon at the end of the statement. 
    System.out.println("Hello, World");
  • Error: "reached end of file while parsing" or "')' expected"
    Cause: Unmatched parentheses, brackets, or braces
    Fix: Ensure all opening parentheses, brackets, and braces have matching closing ones. 
    if (condition) {
                // code
            }
  • Error: "cannot find symbol" or "variable x is not defined"
    Cause: Using undeclared variables
    Fix: Declare the variable before using it. 
    int x = 5;
            System.out.println(x);
  • Error: "cannot find symbol" (for methods) or "cannot resolve symbol" (for variables)
    Cause: Incorrect method or variable names (case-sensitive)
    Fix: Check the spelling and capitalization of method and variable names. 
    String message = "Hello";
            System.out.println(message); // Not Message or MESSAGE
  • Error: "main method not found in class"
    Cause: Forgetting to declare the main method
    Fix: Include the main method in your class. 
    public class MyClass {
                public static void main(String[] args) {
                    // Your code here
                }
            }
  • Error: "incompatible types" or "possible lossy conversion"
    Cause: Incorrect data type assignments
    Fix: Ensure you're assigning values to variables of the correct type, or use explicit casting when necessary. 
    int x = 5;
            double y = 2.5;
            int z = (int) y; // Explicit casting
  • Error: "ArrayIndexOutOfBoundsException"
    Cause: Trying to access an array index that doesn't exist
    Fix: Ensure array indices are within the valid range (0 to length-1). 
    int[] arr = {1, 2, 3};
            System.out.println(arr[2]); // Last valid index is 2
  • Error: "NullPointerException"
    Cause: Trying to use a reference variable that points to a null object
    Fix: Initialize objects before using them, or check for null before accessing object methods or properties. 
    String str = null;
            if (str != null) {
                System.out.println(str.length());
            }
