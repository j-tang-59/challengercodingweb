Scratch 1 Class - Guided Exercise

Lesson 3 Guide

Vocabulary:

  • Animation: Bringing a sprite to life by creating the illusion of movement.
  • Costume: The different appearances or "outfits" a sprite can wear.
  • Backdrop: The background image that sets the stage for your story or game.

Blocks Used/Introduced Today:

  • This block changes the sprite's costume to a specific one.

  • This block changes the sprite's costume to the next one in the sequence.

  • This block pauses the action for a specified number of seconds.

Exercise 1: Animating a Sprite

Grab a sprite with multiple costumes.


Use the "Switch Costume" block to change its appearance.


Follow it with a "Wait" block to control the speed of the change.


Add the "Next Costume" block to create a simple animation.


Try running your code to see the sprite come to life!

Exercise 2: Creating a Story

Pick a sprite to be your main character.


Set up a scene with the "Switch Backdrop" block.


Make the sprite talk using the "Say" block.


Add costume changes to show emotions or actions.


Switch scenes with the "Next Backdrop" block.

Exercise 3: Bringing it All Together

Set up your first scene with a backdrop and sprite.


Use "Switch Costume" and "Next Costume" to animate your sprite.


Add dialogue with "Say" and "Think" blocks.


Move between scenes with the "Next Backdrop" block.


Create a mini-movie by combining all these elements!

Homework:

  • Exercise 1: Create Your Own Animation

    Animate a sprite by switching its costumes. Experiment with different speeds and costume sequences.

  • Exercise 2: Build a Story

    Use sprites, costumes, and backdrops to create a short story. Include dialogue and scene changes.

  • Exercise 3: Combine Animation and Storytelling

    Create a mini-movie that combines everything you've learned about animation and storytelling in Scratch.
