Exercise 1: Introduction to Conditionals in Java

Conditionals are statements in Java that allow you to execute certain blocks of code depending on whether a boolean condition is true or false.

In programming, a conditional statement evaluates to either true or false. If your conditional statement evaluates to true, the associated block of code will run. If not, the program will skip it or execute an alternative block of code.

Think of it as your mom telling you to finish your chores. If you finish your chores, you will get your allowance. If you don’t, you won’t get anything.

boolean choresComplete = true;
    if (choresComplete) {
        // receive allowance
    } else {
        // no allowance
    }

In Java, there are three types of conditional statements:

If statement: The if statement begins the conditional check. If the condition inside the parentheses is true, the code inside the block runs.

Else if statement: This is the middle statement that checks another condition if the previous if condition was false. It is optional and can be used multiple times.

Else statement: This ends the conditional chain. If none of the above conditions were true, the code inside the else block will run.

int score = 75;
    
    if (score >= 90) {
        System.out.println("A grade");
    } else if (score >= 80) {
        System.out.println("B grade");
    } else if (score >= 70) {
        System.out.println("C grade");
    } else {
        System.out.println("Failed");
    }

Additionally, Java uses boolean expressions in conditions. These expressions evaluate to either true or false. Common boolean expressions include:

  • == (Equal to)
  • != (Not equal to)
  • > (Greater than)
  • >= (Greater than or equal to)
  • < (Less than)
  • <= (Less than or equal to)

Exercise 2: Working with Nested If Statements in Java

In Java, a nested if statement is an if statement inside another if or else statement. Nested if statements allow you to check multiple conditions in a more complex decision-making process.

Imagine you’re baking a cake and you want to check if you have the right ingredients and enough time to bake. You would first check if you have all the ingredients, and if that’s true, you would then check if you have enough time to bake the cake.

boolean hasIngredients = true;
boolean hasTime = false;

if (hasIngredients) {
    if (hasTime) {
        System.out.println("You can bake the cake!");
    } else {
        System.out.println("You don't have enough time to bake the cake.");
    }
} else {
    System.out.println("You don't have the necessary ingredients.");
}

Notice how the second if statement is nested inside the first if statement. This structure allows for more detailed condition checks.

Exercise 3: Using Logical Operators in Java

Logical operators in Java are used to combine multiple conditions in a conditional statement. The most common logical operators are:

  • AND (&&): The code inside the if statement will execute only if both conditions are true.
  • OR (||): The code inside the if statement will execute if at least one of the conditions is true.
  • NOT (!): The code inside the if statement will execute if the condition is false.
int age = 20;
boolean hasTicket = true;

if (age >= 18 && hasTicket) {
    System.out.println("You can watch the movie.");
} else {
    System.out.println("You cannot watch the movie.");
}

In this example, both conditions must be true for the person to watch the movie. If either condition is false, they cannot watch the movie.
