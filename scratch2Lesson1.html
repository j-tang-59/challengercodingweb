Scratch 2 Class - Guided Exercise

Lesson 2: Pong Game (1 Player)

Overview:

In this lesson, you will create a basic 1-player Pong game in Scratch. The player controls a paddle on the left side of the screen using the mouse, while a bouncing ball tries to pass the paddle. Your goal is to keep the ball in play and prevent it from passing your paddle.

The ball will bounce off the edges of the screen and your paddle. You will score points by keeping the ball in play. If the ball passes your paddle, the game will end.

Check out the final project below!

Mini-Lesson 1: Setting Up the Stage and Paddle

Step 1: Setting the Stage

We’ll start by creating a simple playing field for the Pong game.

Go to the "Backdrops" tab and select or create a simple backdrop to represent the playing field. You can use the built-in "Blue Sky" or create a plain-colored background.

Step 2: Creating the Paddle

Next, we’ll create the player’s paddle. How do you think we should design it?

Go to the "Sprites" tab and create a new sprite. Draw a simple rectangle, name it "PlayerPaddle," and position it on the left side of the screen.

Mini-Lesson 2: Moving the Player's Paddle with the Mouse

Step 1: Player-Controlled Paddle

We’ll control the player's paddle using the mouse. How can we set this up in Scratch?

Use the forever block and place the go to y: [mouse y] block inside it. This will allow the paddle to follow the mouse vertically on the screen.

Mini-Lesson 3: Ball Movement

Step 1: Creating the Ball

Now, we’ll create the ball that bounces between the paddles. How can we make the ball move in Scratch?

Draw a small circle sprite and use the move [ ] steps and point in direction [ ] blocks to make the ball move.

Step 2: Bouncing the Ball

How can we make the ball bounce when it hits the paddle or the edges of the screen?

Use the if on edge, bounce block to make the ball bounce off the screen edges. For the paddle, use the if touching [PlayerPaddle] block to make the ball bounce off the player's paddle.

Mini-Lesson 4: Scoring and Game Over

Step 1: Adding a Score Counter

How can we track the player’s score when the ball hits the right side of the screen?

Use a variable called PlayerScore and increase the score each time the ball passes the right edge of the screen.

Step 2: Game Over

How should we end the game when the player misses the ball?

When the ball passes the player’s paddle, broadcast a Game Over message and stop all scripts.

Homework:
