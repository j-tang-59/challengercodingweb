Exercise 2: While Loops

                
While loops are the simplest type of loops. Like an if statement, they have a condition. If the condition is true, the while loop will continue running.

                
The computer checks if the condition is still true after it finishes the current round of the loop.

                
Example: If you had a while loop that prints "Hi" and then prints your name, the code would only check if the condition is true after it prints your name.

                
You can also use:

                
    
                    
  • break: To break out of the loop early before it naturally ends.
    • 
                    
  • continue: To skip the rest of the current loop iteration and move to the next one.
    • 
                    
  • True: A Boolean variable to create a loop that never ends.
    • 
                

                
While Loop Syntax:

                
while condition:
    # code to be executed
    if some_condition:
        break  # optional

Exercise 3: For Loops

                
For loops run the code for a certain range or number of times. The range() function is often used to provide the program with a range within which it will run the code.

                
range syntax:

                
range(STARTING, ENDING)

                
The STARTING is the first number in the range, and the ENDING is the last number plus one. So, if you want the range to be 0 to 10, you will write:

                
range(0, 11)

                
The syntax of for loops can be tricky:

                
for VARIABLE in RANGE:
    # code to be executed

                
The VARIABLE represents the current item in the range, and the RANGE can be anything with a length or a range, such as a list, a string, or the range function.

Exercise 4: How For Loops Work

                
For loops start with the first item in the list or range and set the VARIABLE to it. The code inside the loop is then executed for that value. The loop then moves to the next item and repeats the process.

                
Example:

                
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

                
This loop will print each fruit in the list until all items have been processed.
