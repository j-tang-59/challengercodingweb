Python Week 2: Overview

This lesson introduced key concepts regarding variables and their role in storing information. We also explored the three main types of variables in Python and different operations that can be performed on them.

Key Topics Covered:

  • Introduction to Variables
  • Data Types: String, Integer, Float
  • Declaring and Redefining Variables
  • Operations on Integer and Float Variables
  • String Concatenation

Definitions:

Variables:

Variables are used to store information in memory. Think of a variable like a labeled box, and the information inside the box is what the computer remembers. Variables get erased and replaced each time you rerun your program.

Data Types:

Data types represent the kinds of information you can store in variables. Python allows a flexible system where variables can switch between types as needed. We covered the following:

  • Strings: Sequences of characters, including text, surrounded by quotation marks (" " or ' ').
  • Integer: Whole numbers without decimals.
  • Float: Numbers with decimal points.

Examples:

Declaring a String: name = "Aadi Saraf"
Declaring an Integer: num = 10
Declaring a Float: pi = 3.14159

String Concatenation:

Strings can be added together using concatenation. This combines multiple strings into one longer string.

Example: greeting = "Hello, " + "world!" will produce "Hello, world!"

Operations on Variables:

Python allows you to perform operations on integers and floats, such as addition, subtraction, multiplication, and more.

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
  • Floor Division: // (Removes decimals)
  • Modulo: % (Finds the remainder)

Redefining Variables:

Variables can be redeclared at any time. For example:

Initial: num = 10
Redeclaration: num = 20

Advanced Printing:

To insert variables into a string during a print() statement, use an f-string. It lets you include variables within a string by surrounding the variable name with curly brackets { }.

Example: print(f"My favorite number is {favNumber}")
