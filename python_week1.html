Python Week 1: Overview

This lesson focused on preparing your development environment and grasping basic concepts that are foundational for programming with Python.

1. Setting Up VS Code:

We began by installing Visual Studio Code (VS Code), a popular editor for writing code. If you need to set up VS Code, follow the instructions provided on this guide.

2. Understanding Files and Terminals:

  • File Names and Endings: We learned about saving files with the proper extensions (e.g., .py for Python).
  • The Terminal: This tool shows the output from your scripts, such as print statements, directly on your screen.

3. Introduction to Print Statements:

The print() function is used to output text to the terminal. It's a basic but powerful tool for beginning programmers.

print("Hello, World!")

Using quotes tells Python that the enclosed text is a string, a datatype used for storing text.

4. First Coding Experience:

We practiced using simple print statements to display messages in the terminal.

Key Concepts from this Lesson:

  • Text-Based Languages: Unlike block coding, text-based languages use text to instruct the computer.
  • Functions: Reusable pieces of code that perform actions, such as displaying text.
  • Terminal: Where the results of your code are displayed.

By the end of this lesson, you should be comfortable with setting up your environment and writing basic Python code.
