Exercise 2: Data Types

                
A string is all the words and letters that are not functions and operations. Strings must be surrounded by either quotation marks or single quotes. Remember the “Hello World” from last class? That was a string too!


                
string = "hello"

                
good_string = "I love Python"

                
Note that you can also put numbers in the strings, as long as they are inside the quotation marks! However, these examples are NOT strings.

            
                
12, 10, 1.23, 3.14159

            
        
                
The Integer is the simplest of the three main data types. It is simply a number, without a decimal point.

       
                
1, 10, 343, 21

                
Not an example:

                
hello, 1.0, 2.3, 3.14159, 2.10

                
We can perform functions like adding, subtracting, division, and basically all the math functions you can think of with these.

        
                
Float: Floats are the final of the three main data types. The Float is simply a number that has a decimal point!

                
                
1.124, 3.14159, 1.20, 2.0

                
Not an example:

                
hello

                
Floats can be added, subtracted, and all the other mathematical operations too!
