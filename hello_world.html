Exercise 1: Say hello

The very first program you will learn to write is hello, world. In Python, the command to display hello, world on the screen is simply:

print('hello, world')

print is a special command to tell the computer program to show whatever is inside the parentheses.

The quoted text, 'hello, world', is called a string. Think of a string as a sequence of characters that are strung together in order.
